# 스타일 규칙

## 톤 앤 매너
- 단정하되 친근하게 — 무섭거나 무거운 인상 금지
- 전문성은 구체적 용어로 드러내되, 풀어서 설명
- 공감형 어투: "~하는 사람 있죠?", "이런 경험 있으신가요?"
- 마무리는 긍정적 인사이트 또는 실용적 조언으로

## 제목 작성 원칙
- 숫자 포함 시 클릭률 상승 ("3가지", "TOP 5", "딱 2가지")
- 의문문 또는 대상 지정 ("○○인 사람은?", "당신의 ○○은?")
- 25자 초과 금지 (카드에 잘려 보임)

## 금지 사항
- 특정 날짜 고정 콘텐츠 ("오늘", "이번 주" 등) → 영구 재활용 불가
- 정치·종교·의료 단정 발언
- 과도한 확신체 ("반드시", "무조건", "100%")
- 지나친 부정 예언 ("망한다", "불행해진다")
- 히스토리에 이미 있는 title 또는 angle

## 언어
- 기본 한국어
- 전문 용어 첫 등장 시 한자 병기: 오행(五行)
- 해시태그는 한국어·영어 혼용 가능
